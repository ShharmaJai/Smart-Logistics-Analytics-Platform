# Smart Logistics Management & Analytics Platform

## Project Overview
A centralized logistics analytics dashboard built to help management monitor 
shipments, track courier performance, analyze costs, and detect cancellation patterns.

## Tech Stack
- Python
- MySQL
- Streamlit
- Pandas

## Dataset
6 datasets with 70,000+ shipment records including shipments, tracking logs, 
courier staff, routes, warehouses and costs.

## Dashboard Features
- Home Page with KPI cards
- Shipment Search and Filtering
- Delivery Performance Analysis
- Courier Performance Analysis
- Cost Analytics
- Cancellation Analysis
- Warehouse Insights

## How to Run
1. Install required libraries: pip install streamlit mysql-connector-python pandas
2. Set up MySQL database: logistics_db
3. Run: streamlit run logistic_app.py
